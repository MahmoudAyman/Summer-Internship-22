Electronics Shelf

Ideation

Working with electronics can be a bit messy, espesially when it's a busy day and a lot a people are using the electronics bench. Thus we want to improve the existing setup, by building a shelf for the electronics bench. We will also build storage drawers to store the different components we have in our inventory. We want the shelf to be modular and scalable, we want to be able to add different features/parts in the future and integrate more storage boxes easily when we have more components to store. The image above, shows a free hand sketch showing main features of the project. The goal is to cooperatively design and manufacture this shelf.
