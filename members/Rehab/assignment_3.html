Assignment 3: Electronics Container

Follow the light, it will take you home ")

Requirements: contains 16 drawer (4*4) each one is 40*40*70 mm

Step 1: Free hand sketch

After collecting some different ideas from different resourses (which are Ayman's drawing). Finally, we can choose our one.

Step 2: Drawing

Open any 2D CAD tool (as SolidWorks, AutoCAD, Fusion, .. ) and JUST KEEP DRAWING :)

Then save as DXF format :)

Another note: You can draw it and assume any missing dimension.

Step 3: From DXF to U file

Open RDWorks software.

file --> import --> select your .dxf file

adjust the speed and power to be suitable with 6 mm and 3 mm wood thickness .

Finally, export the machin format file (U file) from SaveToUFile button.

Step 4: Cutting Now

The final step is uploading the U file into a flash memory, read this flash memory by the machine, and copy it to the machine memory.

Adjust the new origin of the machine to be suitable with the existing wooden sheet. Check the working area from frame button.

Finally, press start button ").

Step 5: Enjoy with your container

OOPS! .. Don't forget to draw your drawer ").
