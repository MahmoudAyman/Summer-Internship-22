Assignment 2: Stationery Organizer Desk

The idea of assigment is to Use the laser cutter to make a useful tool, of your design, that can be used in the lab so I choose to make Stationery Organizer Desk

step1:

I started with brainstorming to see how I can make the design of the shape so I started with hand sketching to imagine the dimensions.

step2:

After making hand sketching I started to use SolidWorks to draw parts that are seen in the page above after that I started to make assembly to put the parts together

From the button you can see the Solidworks files

step3:

The next step is to use the RD Works Program to prepare the files to be ready for printing in the laser cutting machine so I import the files from Solidworks as DFX files

After I import the files I start to setup the power and speed that will used in machine and here as I only want to cut in wood

Speed =25 mm/s

Power= 70

From the button you can see the Rd files

step4:

The next step is to start to make the machine reads the rd files from RD Work Program and start to cut the files.

The wood thickness that I use is 3 mm
