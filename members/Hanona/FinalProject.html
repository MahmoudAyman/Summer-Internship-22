Final Project | The Claw Machine

Soo, The idea of my final project is making my own version of the claw machine and scalable one using simple claw mechanism and electronics

It will be the same concept of the real version of the claw machine but the prizes will be different somehow as well as the design

First | Description

The Game:-

1. The waiting phase, which is the default state of the machine waiting for people to come and play, home position for the claw, lights are one differenlty and music backgroung
2. Coin inserted, when someone insert a coin into the coin area and light will be green and there will be a sound effect of a coin being inserted to make it ore fun
3. Press start button, to start the game u should press the button and then start using the joystick, then there will be a game mysic at the background
4. press start button again (or time is over), when u place the claw in the desired place u should press the button again to make the claw drop

There will be a timer for 20 sec to press the button again, if it is not pressed, there will be no ore joystick contact and the claw automatically will be dropped after 20 seconds
