Circuit Explination

The main concept of this circuit is changing the tone of the sound by changing the frequency. The frequency value is dependent mainly on the values of resistors and capacitor as shown in this equation:

Frequency (f) = 1.44 / ( (R1 + 2 * R2) x C1 )

we can replace the value of R1 (Mostly it is 1K ohm) then set the value of C1 (Mostly 100nf) and rearrange the equation to calculate the required value of R2 to get the required frequencies

Frequency (f) = 1.44 / ( (1000 + 2 * R2) x 10-7 )
Resistance (R2) = ( ( 1.44 x 107 ) / f ) - 500
